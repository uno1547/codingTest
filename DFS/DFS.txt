DFS 알고리즘
depth first search 깊이 우선 탐색
탐색 : 많은데이터중 원하는 데이터를 찾는 알고리즘
그래프 : 자바스크립트에서 그래프 2차원배열 (인접 리스트 형식) 을 사용한다.

그래프/트리에서 모든 노드를 한번씩(****) 탐색하기 위한 알고리즘이 깊이우선탐색이다!!
DFS는 스택으로 구현할수있다.

1. 스택으로 구현가능
- 스택에 시작점 push, 방문처리
- 스택 최상단 노드의 인접노드가 존재하는지 check
- 있다면, 인접노드를 스택에 push, 방문처리 (한번에 여러개 넣는개 아니라 한개씩넣으면서 다시 검사)
- 없다면, 최상단 노드 pop
- 스택에서 pop, 인접정점 push
-

2. 재귀함수로도 구현가능 이게 스택보다 구현이 간단하다고함!!

- 방문표시(시작)
- 인접노드있다면 재귀호출, 방문표시

특징 : 탐색속도가 BFS보다 느리다. but구현방식이 간단해 (재귀함수로 하면) 
큐라이브러리 사용불가 (= BFS불가? 큐 ) 일경우 사용
**** 트리의 순회, 점화식 같이 특정상황에서 유리함
**** 트리에서 최단경로를 구할 때 사용가능하다

그래프 2차원배열에서 0번째 줄은 사용하지않는것이 카운팅에서 편리하다(헷갈리지않는다)
